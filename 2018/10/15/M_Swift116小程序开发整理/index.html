小程序

发表于 2018-10-15

api地址：https://developers.weixin.qq.com/miniprogram/dev/component/

1.小程序配置 app.json
app.json 是当前小程序的全局配置，包括了小程序的所有页面路径、界面表现、网络超时时间、底部 tab 等。
{
“pages”:[
“pages/index/index”,
],
“window”:{
“backgroundTextStyle”:”light”,
“navigationBarBackgroundColor”: “#fff”,
“navigationBarTitleText”: “WeChat”,
“navigationBarTextStyle”:”black”
}
}
2.pages小程序页面
小程序页面包含4个部分
wxml： 是用来描述当前这个页面的结构
wxss：用来描述页面的样子
js：通常是用来处理这个页面和用户的交互
json：页面配置，页面中配置项会覆盖 app.json 的 window 中相同的配置项。
3.数据绑定js交互

Page({
clickMe: function() {
this.setData({ msg: “Hello World” })
}
})
4.小程序的启动
默认第一个配置就是启动页面
5.页面 Page
data Object 页面的初始数据
生命周期函数等等
6.页面跳转
打开新页面 调用 API wx.navigateTo
页面重定向 调用 API wx.redirectTo
页面返回 调用 API wx.navigateBack
Tab 切换 调用 API wx.switchTab
重启动 调用 API wx.reLaunch
7.组件
view、label、text等等
8.微信开放api

Android Stdio 快捷键
web百度地图使用
