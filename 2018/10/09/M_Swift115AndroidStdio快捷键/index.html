Android Stdio 快捷键

发表于 2018-10-09

快速定位类名：Cmd + O
查看本类中方法：Cmd + Fn + F12
查看方法被调用的位置：Option + Fn + F7
项目内全局搜索：Cmd + Shift + F
格式化代码：Cmd + Option + L
快速定位所有类名及其中的方法名：Cmd + Option + O
清除无效包引用：Ctrl + Option + O（自动清除无用包，加载有用包，Android Studio基本自动执行了）
显示Version Control面板：Cmd + Shift + 9

复制当前行：Cmd + D
查找：Cmd + F
构造方法，set get方法等生成的快捷菜单：Cmd + N
查找+替换：Cmd + R
删除行：Cmd + Delete
注释代码(//)：Cmd + /
导包（提示错误解决方案）：Option + Enter
重命名变量：⇧F6
