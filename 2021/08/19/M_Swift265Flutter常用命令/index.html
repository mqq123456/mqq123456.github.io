Flutter常用命令

发表于 2021-08-19 更新于 2022-08-24

flutter run -d macos
flutter doctor:检查flutter状态
flutter emulators:获取模拟器列表
flutter devices:获取所有真机设备列表包括iOS模拟器
flutter run -d :运行指定模拟器或者真机
6.flutter build apk:打包apk
flutter config -xx:配置打包等信息。
flutter channel:查看flutter sdk的所有分支
flutter packages get:安装了相关插件
flutter packages upgrade:更新flutter项目所有依赖包，不包括flutter sdk
flutter create project_name
flutter upgrade
flutter analyze 分析项目代码
flutter clean 清空build
iOS打包framework
flutter build ios-framework –output=../flutter_framework
flutter build aar android 打包aar
添加插件
flutter pub add flutter_markdown
// 打开模拟器
open -a Simulator
// 开发模式下的自动生成
flutter pub run build_runner build
// 非空安全
flutter run –no-sound-null-safety
//
flutter pub run build_runner build –delete-conflicting-outputs

当然你也可以在打包时通过 flutter build web –web-renderer html –release 之类的配置强行指定渲染模式。
// 创建plugin
flutter create –template=plugin –platforms=android,ios -i objc -a java face_verify_plugin
