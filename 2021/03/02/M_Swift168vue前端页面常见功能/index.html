前端页面常见功能

发表于 2021-03-02 更新于 2021-04-12

1.echarts图标（柱状图、折线图、饼状图等）
参考echarts官网
https://echarts.apache.org/zh/index.html
2.echarts和百度地图结合展示（地图覆盖物等）
参考百度地图js api 及echarts地图组建
https://echarts.apache.org/examples/zh/index.html
3.列表数据导出excel
后端实现，前端采用axios 支持excel格式导出
responseType: ‘blob’,
4.word编辑下载及转pdf预览
目前项目采用json格式存储word，前端解析json数据展示成word文档样式，编辑就是修改json数据，数据采用v-mode模式双向绑定输入框，实现动态修改。
导出采用docxtemplater前端定义word模版格式，将数据塞进word模版中
echarts采用前端生成，转成图片存放如前端word模版中
pdf预览采用将前端word模版发送给后端，后端通过aspose-words工具转成pdf存储进文件服务器
5.excel编辑、下载
参考el-table的编辑功能
